# AI Smart Recommender System for Students

This project is an AI-powered smart recommender system designed to help students
discover personalized learning paths, course recommendations, and career guidance
using hybrid recommendation techniques.

## Features
- Hybrid recommendation system (content-based + collaborative)
- Personalized student dashboards
- Modern React + TypeScript frontend
- Scalable architecture

## Tech Stack
- Frontend: React, Vite, TypeScript, Tailwind CSS
- Backend: Node.js, Express (optional integration)
- Database: MongoDB
- AI/ML: Hybrid recommendation models
